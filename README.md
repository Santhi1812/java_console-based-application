**Library Rental System (Java Console Application)**

**Description**
The Library Rental System is a Java-based console application designed to manage book rentals in a library. It allows users to view available books, issue and return books, and maintain basic library records through a simple text-based interface.
**Features:**
Add, view, and manage book details
Issue and return books with availability tracking
User-friendly menu-driven console interface
Handles basic input validation and error checking
**Technologies Used:**
Java
OOP concepts (Classes, Objects, Encapsulation)
Collections (ArrayList / HashMap)
Console I/O
**Future Enhancements:**
Add user authentication
Integrate file or database storage
Implement fine calculation for late returns
